# Admin Panel Data

Раздел отвечает за таблицу админ-панели: операторы/админы, анкеты, дневные суммы, подарки, цвета и комментарии.

## Где хранится сейчас

В `data.json`:

- `agencyBonusLedger` - строки балансов/бонусов из агентской панели.
- `assignmentHistory` - история, по которой анкета попадает в месяц даже после снятия.
- `adminPanelCellColors` - цвета ячеек по месяцам.
- `adminPanelCellComments` - комментарии к ячейкам/строкам.
- `users[].adminPanelSource` - источник/пометка оператора.
- `salaryRates`, `salaryFeePercent` - связанные настройки расчетов.

Будущая структура хранения баланса по месяцам описана в `BALANCE_STORAGE.md`.

## Основная логика

- День считается через business day, а не просто календарную дату.
- Месячная таблица собирается по выбранному месяцу.
- Активные анкеты считаются в `LADIES`.
- Исторические снятые анкеты должны оставаться строками месяца, если были назначены в этом месяце.

## Что можно менять в рамках Admin Panel

- отображение таблицы;
- расчет строк таблицы;
- цвета и комментарии;
- загрузку балансов из агентской панели;
- сортировку строк.

## Что нельзя трогать без отдельной задачи

- отправку писем;
- вложения фото/видео;
- алгоритм Inbox/Sent;
- запуск Dream страницы;
- логины/пароли;
- ручное изменение `data.json`.

## Проверка после изменений

- `/ ?adminPanel=1` открывает таблицу;
- текущий месяц использует текущий день, а не первое число месяца;
- снятая анкета видна как неактивная строка месяца;
- цвета и комментарии сохраняются;
- `LADIES` считает только активные анкеты, если специально не задано другое правило.
