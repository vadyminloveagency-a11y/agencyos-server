# Sent Data

Раздел отвечает за отправленные письма и исходящие элементы диалога.

## Где хранится сейчас

Сейчас исходящие данные в основном связаны с:

- `profiles[profileId].workspaceInbox[].conversation` - диалог, где могут быть исходящие сообщения;
- данными, полученными из Dream Singles при синхронизации Sent;
- рабочим состоянием фронтенда в `public/workspace.js`.

Если в будущем Sent будет вынесен отдельно, целевое место:

- `profiles[profileId].workspaceSent`.

## Правила Sent

- Sent может грузиться отдельно от Inbox.
- Sent не должен очищать Inbox.
- Sent не должен менять назначения анкет.
- Повторная загрузка должна обновлять существующие письма, а не плодить дубли.

## Что нельзя трогать без отдельной задачи

- логины/пароли;
- Admin Panel;
- Inbox-ключи;
- Dream gallery media.

## Проверка после изменений

- исходящие письма видны в диалоге;
- новые исходящие не дублируют старые;
- Inbox продолжает работать.
