# Settings Data

Раздел отвечает за настройки CRM, настройки пользователя, переводчик, Google Drive и внешние ключи.

## Где хранится сейчас

В `data.json`:

- `translator` - настройки переводчика.
- `translationCache` - кеш переводов.
- `users` - настройки пользователей и агентские доступы.
- `profiles[].googleDrive` или похожие поля - настройки Google Drive, если заданы.
- `profiles[].credentials` - зашифрованные Dream credentials.

В корне проекта:

- `.credential-key` - ключ шифрования.
- `.env` - локальные секреты, если используется.

## Правила

- Секреты нельзя показывать в UI и логах.
- Кеш переводов можно чистить только осознанно.
- Настройки Google Drive не должны менять Inbox, Sent или Admin Panel.

## Что нельзя трогать без отдельной задачи

- `passwordHash`, `passwordSalt`;
- `.credential-key`;
- реальные пароли анкет;
- `data.json` вручную.

## Проверка после изменений

- переводчик работает;
- секреты не выводятся в API;
- Google Drive настройки сохраняются;
- вход в CRM не ломается.
