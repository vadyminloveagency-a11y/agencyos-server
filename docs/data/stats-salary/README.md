# Stats And Salary Data

Раздел отвечает за статистику, проценты, зарплаты и финансовые итоги.

## Где хранится сейчас

В `data.json`:

- `agencyBonusLedger` - финансовые строки из агентской панели.
- `salaryRates` - шкала процентов.
- `salaryFeePercent` - процент комиссии/сайта.
- `assignmentHistory` - нужно для понимания, кому принадлежала анкета в конкретный день.

## Правила

- Финансы считаются по дате строки и business day.
- Зарплата зависит от баланса, комиссии и процентной шкалы.
- Историческое назначение анкеты важно для правильного расчета прошлых дней.

## Что нельзя трогать без отдельной задачи

- Inbox/Sent;
- отправку писем;
- Dream media;
- пароли;
- текущие назначения без проверки истории.

## Проверка после изменений

- дневные итоги совпадают с ledger;
- месячные итоги совпадают с суммой дней;
- снятые анкеты считаются за период, когда они были назначены;
- изменение процентов не меняет Inbox или Dashboard.
