# Mail And Media Data

Раздел отвечает за отправку писем, рассылку, фото и видео в письмах.

## Где хранится сейчас

В `data.json` внутри анкеты:

- `profiles[profileId].workspaceMediaGallery` - фото/видео, доступные для письма.

Во вложениях письма используются:

- `galleryId` - ID фото из Dream gallery.
- `videoGalleryId` - ID видео из Dream video gallery.
- `source` - источник вложения, например Dream gallery.
- `url`, `thumbUrl`, `title`, `type`.

В браузерном расширении:

- логика прямой отправки письма и прикрепления Dream gallery media находится в расширении `dreamsearch_man_v14`.

## Правила отправки

- Письмо выбранному мужчине отправляется не на случайный адрес, а по выбранному диалогу/мужчине.
- Фото и видео должны прикрепляться как Dream gallery вложения, если они выбраны из Dream.
- Обновление фото/видео должно использовать существующую логику Dream gallery.

## Что нельзя трогать без отдельной задачи

- Admin Panel;
- `assignmentHistory`;
- зарплаты и балансы;
- удаление Inbox/Sent;
- формат паролей.

## Проверка после изменений

- письмо отправляется выбранному мужчине;
- фото уходит с `galleryId`;
- видео уходит с `videoGalleryId`;
- без выбранных вложений письмо отправляется текстом;
- обновление медиа не требует ненужного открытия Dream страницы, если прямой путь доступен.
